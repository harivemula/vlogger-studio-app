# Vlogger Studio App

## Overview
Vlogger Studio is a comprehensive video creation platform designed for content creators, vloggers, and video producers. This website showcases the features, pricing, and information about the Vlogger Studio application.

## Website Pages

### Main Pages
- **Homepage** (`index.html`) - Main landing page with hero section, feature highlights, testimonials, and calls-to-action
- **Features** (`features.html`) - Detailed breakdown of all platform capabilities including recording, editing, templates, and publishing
- **Pricing** (`pricing.html`) - Three subscription tiers (Starter, Creator, Studio) with feature comparisons
- **About** (`about.html`) - Company mission, values, story, and statistics
- **Download** (`download.html`) - Download options for Windows, macOS, and Linux with system requirements
- **Contact** (`contact.html`) - Contact form and support information

### Legal Pages
- **Privacy Policy** (`privacy-policy.html`) - Comprehensive privacy policy including GDPR and CCPA compliance
- **Terms and Conditions** (`terms-and-conditions.html`) - Complete terms of service

## Local Development

To view the website locally:

```bash
# Navigate to the project directory
cd vlogger-studio-app

# Start a local web server
python3 -m http.server 8080

# Or use any other local server
npx serve .

# Open in browser
# http://localhost:8080
```
